# 武林外传答案之书 — 开发日志

## 2026-07-04 — 项目启动

### 决策记录

1. **技术栈**：React 18 + Vite + react-pageflip + Zustand
   - 原因：用户指定 React/Vite，react-pageflip 是最成熟的 React 翻书库
   - 后续小程序迁移：先用 WebView 套壳，必要时 Canvas 重写翻页

2. **数据管线**：Python (zipfile + xml) 解析 .docm → 正则提取 `角色：台词` → LLM 分类 → JSON
   - 原因：python-docx 不支持 .docm 格式，直接用 zipfile + XML 解析
   - 分类由 LLM 辅助完成，人工抽查

3. **分类体系**：丧类/躺平类/毒舌类/扎心类/哲理类/治愈类/爱情类/职场类/人生类
   - 存储在 categories.json 中，不硬编码

4. **少用硬编码**：分类名、路径、主题色均走配置文件

### 数据源
- 剧本：武林外传电视剧版剧本.docx（49544段，80回，.docm格式）
- 参考：武林外传-答案之书.docx（34条手选句子）
- 参考：80句台词答案之书武林外传.docx（已有分类示例）

### 2026-07-04 网页开发记录

**已完成：**
- Vite + React 18 项目搭建
- react-pageflip 2.x 集成，3D翻书效果
- Zustand 状态管理（cover ↔ book 切换、随机翻页）
- 封面页：靛蓝古书封面、金色竖排标题、脉动光晕、飘浮粒子
- 翻书页：硬封面+扉页+20条答案+封底，带随机翻页和返回功能
- 答案卡片：分类标签+引号正文+角色署名+中文页码
- 古风主题：宣纸米黄/墨色/朱砂红/金箔色彩体系，CSS变量管理
- 示例数据：20条武林外传风格答案（sample_answers.json）
- 禁用硬编码：分类名→categories.json，主题色→theme.css变量

**技术细节：**
- react-pageflip 使用 DOM-based 方案（HTMLFlipBook），支持触摸手势
- 翻页回调 onFlip 同步当前页码到 Zustand
- 随机翻页通过 flipBookRef.pageFlip().flip(pageNum) 实现
- 小程序迁移：后续用 WebView 套壳，必要时重写翻页
